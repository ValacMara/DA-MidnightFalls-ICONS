# DA-MidnightFalls-ICONS
Icons needed for the Detention Marks addon to work across users. 
1. Download each file.
2. Open :\World of Warcraft\_retail_\Interface\ICONS. If you do not see the ICONS folder, create a new folder and name it ICONS.
3. Copy and paste all 5 icons into the ICONS folder:\World of Warcraft\_retail_\Interface\ICONS
